# Dashboard-de-Vendas
Dashboard de Vendas em Excel

Desenvolvimento de um dashboard de vendas para análise de faturamento, volume de vendas, perfil de clientes e desempenho por estado.

Ferramentas Utilizadas
.Excel
.Power Query
.Tabelas Dinâmicas
.VBA
.Segmentações de Dados

Indicadores:
.Receita Total
.Quantidade de Produtos Vendidos
.Receita por Mês
.Quantidade por Mês
.Receita por Faixa Etária
.Estados com Maiores Receitas
.Top 3 Produtos Mais Vendidos
